# Main Engineering

This folder is used to run RDFO on engineering problems 1-5.

## Main Files
- `run_main_engineer_1_5.m`: main script.
- `ProbInfo.m`: problem settings.
- `RDFO.m`: RDFO algorithm.
- `results/`: saved results.

## Usage
1. Open `run_main_engineer_1_5.m`.
2. Set the problem index.
3. Run the script in MATLAB.
