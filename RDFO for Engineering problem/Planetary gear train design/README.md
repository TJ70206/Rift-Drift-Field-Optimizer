# Planetary Gear Train Design

This folder is used to run RDFO on the planetary gear train design problem.

## Main Files
- `run_Planetary_gear_train.m`: main script.
- `ProbInfo.m`: problem settings.
- `RDFO.m`: RDFO algorithm.
- `results/`: saved results.

## Usage
1. Open `run_Planetary_gear_train.m`.
2. Run the script in MATLAB.
